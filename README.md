# web1-exercise

Repository for exercises and activities from the **Web Programming I** course.

## Course Information

* **Course:** Web Programming I (WEB1)
* **Semester:** 2026.1
* **Program:** ADS (Systems Analysis and Development)
* **Class:** S3 - Night

## About this Repository

This repository contains the exercises and activities developed during the **Web Programming I** course.
Each folder represents a specific activity, practice exercise, or assignment completed throughout the semester.

The goal of this repository is to organize and store the code created while learning the fundamentals of web development.

## Structure

The activities are organized by folders according to each exercise or topic covered in class.

Example:

web1-exercise
└── basic-classes-and-methods
